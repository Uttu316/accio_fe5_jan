type="module"

  • run js file in isolate context
  • js file does not share variables in global space
  • export and import can be done to any variable
  • named export and default export
  • there always only one default export in each file while importing
  • use same name of named export variables give any name to default export variable
  • alias can be done of named exports by using as keyword
  • import all exported variable form a file using * as Anyname and use it Anyname.variable
